## 📌 Overview

**MindMaze** is a JavaFX-based desktop game designed to test and improve cognitive skills like memory, logic, and reaction speed.
The application is divided into multiple interactive rooms, each offering a unique mini-game challenge.

---

## 🎮 Features

* 🧩 Multiple mini-game rooms:

  * **Logic Room** – Problem-solving challenges
  * **Memory Room** – Pattern and recall tests
  * **Maze Room** – Navigation and reflex-based gameplay

* ⏱️ Timer and countdown system

* 📈 Increasing difficulty levels

* 🔊 Sound effects for better interaction

* 🎨 Simple and user-friendly UI

---

## 🛠️ Tech Stack

* **Java**
* **JavaFX**
* **Scene Builder (for UI design)**

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/Srikabilan21/mindmaze.git
```

2. Open in IntelliJ IDEA / any Java IDE

3. Make sure JavaFX is configured properly

4. Run the main class:

```bash
Main.java
```
